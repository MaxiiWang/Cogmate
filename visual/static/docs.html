Brain Visual 文档

Brain Visual 是 SimWorld 的可视化界面，让你通过 3D 图谱、时间线、对话等方式探索知识库。

💡

访问此界面需要有效的 Token。Token 由知识库所有者生成，不同 Token 有不同的权限。

视图类型

🌍 Globe View - 3D 知识图谱

以 3D 球形展示知识节点和它们之间的关系。节点大小反映连接数，颜色表示知识类型。可以旋转、缩放和点击节点查看详情。

🌳 Tree View - 抽象层树形图

展示从具体事实到高维洞察的抽象层级结构。帮助理解知识是如何被归纳和提炼的。

📅 Timeline View - 时间线

按时间顺序展示知识的创建过程。可以看到知识库是如何随时间积累和演化的。

💬 Chat - 对话交互

与知识库进行对话。提出问题，AI 会基于知识库内容给出回答，并引用相关的事实和观点。

Token 权限

不同类型的 Token 有不同的访问权限：

类型 权限 说明
full 完整访问 浏览所有视图 + 无限问答 + 编辑权限
qa_public 公开问答 有次数限制的问答，适合试用
browse_public 仅浏览 只能查看图谱，不能提问

使用方法

访问界面

通过带 Token 的链接访问：

http://your-brain-url:8000?token=YOUR_TOKEN

浏览知识图谱

  1. 进入首页后，点击「🌍 Globe」进入 3D 视图
  2. 拖动鼠标旋转图谱，滚轮缩放
  3. 点击节点查看详情
  4. 使用筛选器按类型过滤节点

对话问答

  1. 点击「💬 Chat」进入对话界面
  2. 输入你的问题
  3. AI 会基于知识库内容回答
  4. 回答中会引用相关的事实来源

知识类型

知识库中的内容按类型分类：

类型 颜色 说明
事件 🟢 绿色 发生过的事情
观点 🔵 蓝色 个人看法和判断
情绪 🟣 紫色 情感和感受
资讯 🟡 黄色 外部获取的信息
决策 🟠 橙色 做出的决定

关系类型

知识之间的关系：

  • 支持 - A 的观点或证据支持 B
  • 反对 - A 与 B 存在矛盾或冲突
  • 延伸 - A 是 B 的延伸或细化
  • 来源 - A 是 B 的信息来源
  • 导致 - A 导致了 B 的发生

获取 Token

Token 可以通过以下方式获取：

  • 知识库所有者直接分享
  • 在 Hub 平台购买
